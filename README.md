# Simple-ChatBot
set Your Open AI Api Key and Secret Key in .env


-------- create a .env file --------
- set your Api Key and Secret Key inside the .env file

------- How to Run Backend -------

- cd chatGPT
- python app.py

------ how to run frontend ------

- npm start



- if you have a backend engine runnging on 8000, you should change it to 8001 or something else in app.py
